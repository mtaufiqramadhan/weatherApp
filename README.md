# weatherApp
A simple Weather App

### Tutorial
* **Get API from** - [OpenWeatherMap](https://openweathermap.org/api)

* **Change** res/values/strings.xml

```
<resources>
    ...
    <string name="API_KEY">YOUR_API_KEY</string>
</resources>
```
